How it works

Five layers.
One command.

01
Ingestion
Upload CSV or connect MySQL / PostgreSQL via Docker Agent. Scheduled sync at 08:00 AM daily.
FastAPI · SQLAlchemy
02
ETL Engine
Remove nulls, deduplicate, normalize types. Quality score 0–100 with full cleaning log.
Pandas · Pydantic
03
AI Analysis
LLM generates insights, anomalies, recommendations, and chart configs from data summary.
Ollama · Gemini · Claude
04
Consulting Layer
McKinsey SCR framework. Impact × Effort matrix. 30/60/90-day roadmap. Design Agent picks theme.
executive_agent · design_agent
05
Export
9-slide PPTX via pptxgenjs. Executive PDF via WeasyPrint. Saved to PostgreSQL history.
Node.js · WeasyPrint · Redis
Capabilities

Built for
enterprise scale.

⚡
Dynamic Design Agent
6 consulting themes × 5 layouts, auto-selected from dataset fingerprint. Same data = same design. Different data = different design.
🧠
McKinsey SCR Narrative
Situation → Complication → Resolution. Conclusion-first headlines. Impact quantified. Top 3 actions ranked by Impact × Effort.
🔄
SQL Auto-Pipeline
Docker Agent connects to your database. Queries on schedule. Generates and delivers reports automatically — no human needed.
🔴
Redis Job Queue
Jobs persist across restarts. Concurrent users supported. 6-hour TTL with automatic cleanup. Falls back to memory gracefully.
🐘
PostgreSQL Audit Trail
Every completed analysis stored permanently. Full job history with quality scores, insights summary, and timestamps.
🔐
Multi-tenant Security
API key authentication per company. Rate limiting via Nginx. DB credentials never leave the company server. HTTPS only.
